# Programa-de-Sauda-o
Programa que imprime a mensagem "Hello, world!"

# Hello World Interativo em Python

## Sobre o programa
Este é um programa simples em Python criado para praticar entrada e saída de dados.  
Ele começa exibindo uma mensagem de boas-vindas, pede o nome do usuário e, ao final, mostra uma saudação personalizada no formato **Hello World Nome!**.

## Como funciona
1. O programa imprime uma mensagem inicial de saudação.
2. Em seguida, solicita que o usuário informe seu nome.
3. Por fim, exibe a mensagem **Hello World** junto com o nome digitado.
